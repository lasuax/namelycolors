# namelycolors

English
=========

A small library that creates unique color hex codes depending on given text.

## Installation

  `npm install namelycolors`

## Usage

	import { namelyColor } from "namelycolors";
    
	let textColor = namelyColor("Doğa");
  
  
  Output should be "#20AD0D"
